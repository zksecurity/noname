Module noname::compiler

source ·
Expand description

This module is a wrapper API around noname. It is important that user-facing features use the functions here, as they attach the correct filename and source code to errors. It does that by transforming our [Error] type into a [miette::Error] type for all functions here. (via the IntoMiette trait that we define here.)
